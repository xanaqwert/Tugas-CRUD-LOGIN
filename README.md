### Installation


Step 1: Download the Source Code

2. Clone the repo
   ```sh
   https://github.com/xanaqwert/Tugas-CRUD-LOGIN.git
   ```
3. Navigate to the project directory:
   ```sh
   cd Login
   ```
4. Make a database in your php my admin for the login & register authentication.
